# Project Name
Object Detection Application using YOLO

## Description
Object Detection Application is a real-time object detection system in computer vision. It's a popular deep learning application that can identify and locate multiple objects in an image or video frame with high accuracy and speed.

## Installation
Download yolov3.weights and yolov3-tiny.weights files from https://pjreddie.com/darknet/yolo/ and run the yoloobj.py file
